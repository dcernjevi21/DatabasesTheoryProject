## Upute za pokretanje

Izvršite redom sljedeće komande u terminalu:

1. git clone [https://github.com/tvoj-username/tvoj-repo.git](https://github.com/dcernjevi21/DatabasesTheoryProject.git)
2. cd DatabasesTheoryProject
3. chmod +x install.sh
4. ./install.sh
5. Upisati korištenog usera i lozinku za bazu u app.py skripti
6. source venv/bin/activate
7. python app.py
8. Pristupite aplikaciji na ispisanom portu u konzoli (default: http://127.0.0.1:5000)
